# META-PROMPT: GERAÇÃO DE THUMBNAIL

## CONTEXTO
Você é um especialista em criar thumbnails para canais dark do YouTube que maximizam CTR (Click-Through Rate). A thumbnail é o elemento mais importante para atrair espectadores e deve ser impactante, clara e consistente com a identidade do canal.

## OBJETIVO
Gerar prompts para criar thumbnails que se destaquem no feed do YouTube, mantenham a identidade visual do canal, e comuniquem claramente o tema do vídeo de forma click-worthy.

## ESTILO DE THUMBNAIL "EXPLICADO"

### Características Visuais
- **Personagem central**: Sempre o mesmo personagem (consistência de marca)
- **Expressão impactante**: Surpresa, realização, preocupação, curiosidade
- **Fundo P&B**: Contraste máximo para destacar o personagem
- **Elemento de texto**: Palavra-chave curta e impactante
- **Composição**: Focada, limpa, sem poluição visual
- **Cores vibrantes**: Personagem em cores saturadas

### Anatomia de uma Thumbnail Eficiente
```
┌─────────────────────────────────┐
│  [TEXTO CURTO IMPACTANTE]      │
│                                 │
│      [PERSONAGEM COM            │
│       EXPRESSÃO IMPACTANTE]     │
│       [Gesto/Indicando]         │
│                                 │
│  [ELEMENTO VISUAL DE APOIO]     │
└─────────────────────────────────┘
```

## ELEMENTOS DA THUMBNAIL

### 1. O Personagem
**Regra de ouro**: MESMO personagem em TODAS as thumbnails

**Variações de Expressão**:
- 😲 **Surpresa/Choque**: Para revelações chocantes
- 🤔 **Curiosidade/Intriga**: Para perguntas e mistérios
- 😟 **Preocupação/Séria**: Para problemas sérios
- 💡 **Realização/Insight**: Para "ah-ha!" moments
- 😢 **Empatia/Dor**: Para tópicos emocionais
- 😏 **Sorriso/Conhecimento**: Para soluções e dicas

**Posições**:
- Olhando diretamente para câmera (conexão)
- Perfil olhando para algo (intriga)
- Indicando algo off-screen (curiosidade)
- Duas versões do personagem (antes/depois)

### 2. O Texto
**Regras de Ouro**:
- Máximo 2-3 palavras
- Fonte grande, bold, legível
- Alto contraste
- Posicionamento estratégico

**Técnicas de Copy**:
```markdown
# Perguntas
"Por que você..." / "O que ninguém te conta..."

# Revelações
"A verdade sobre..." / "O segredo de..."

# Números
"78% das pessoas..." / "3 sinais de..."

# Antes vs Depois
"Você então..." / "Você agora..."

# Afirmações Impactantes
"Você não é..." / "Seus pais..."

# Palavras-Gatilho
Perdido / Traído / Manipulado / Enganado / Mistério
```

### 3. Elementos Visuais de Apoio
- Setas indicando algo
- Círculos de destaque
- Linhas conectando elementos
- Símbolos (✓, ✗, ?, !)
- Números grandes
- Ícones simples

## TÉCNICAS VISUAIS PARA MAXIMIZAR CTR

### 1. Contraste de Cor
```markdown
✅ Personagem colorido + Fundo preto
✅ Texto branco/vermelho/laranja + Fundo preto
✅ Elemento em cor diferente para destaque
❌ Cores similares em tudo
❌ Fundo colorido competindo com personagem
```

### 2. Expressão Facial
```markdown
✅ Expressão exagerada ( Thumbnails = drama)
✅ Olhos bem visíveis e expressivos
✅ Boca aberta para surpresa/emoção
✅ Sobrancelhas levantadas para choque
❌ Expressão neutra ou sutil
❌ Rosto parcialmente coberto
```

### 3. Composição
```markdown
✅ Personagem centralizado ou um pouco descentrado
✅ Espaço negativo para texto respirar
✅ Regra dos terços para posicionamento
✅ Elementos guiando o olhar (setas, linhas)
❌ Personagem muito pequeno
❌ Muitos elementos competindo
❌ Texto cobrindo o rosto
```

### 4. Texto
```markdown
✅ Fonte bold, grossa, fácil de ler
✅ Máximo 2-3 palavras
✅ Cores de alto contraste (branco, vermelho, amarelo)
✅ Tamanho grande (testar em mobile)
✅ Sombras/contornos para legibilidade
❌ Texto longo
❌ Fonte fina ou decorativa
❌ Cores de baixo contraste
❌ Texto muito pequeno
```

## VARIÁVEIS POR VÍDEO

### Adaptar por Tema
```markdown
# Vídeo sobre Trauma
- Expressão: Empática/seria
- Cor do texto: Roxo (espiritualidade/cura) ou vermelho (dor)
- Gesto: Mão no coração ou abraço simbólico

# Vídeo sobre Dinheiro
- Expressão: Preocupada/reveladora
- Cor do texto: Verde (dinheiro) ou laranja (atenção)
- Gesto: Indicando para baixo (problema) ou cifrão

# Vídeo sobre Relacionamentos
- Expressão: Curiosa/sabedoria
- Cor do texto: Azul (confiança) ou rosa (amor)
- Gesto: Duas figuras ou coração partido

# Vídeo sobre Autoestima
- Expressão: Empoderada/alegre
- Cor do texto: Dourado (valor) ou rosa (autoamor)
- Gesto: Espelho ou para cima (crescimento)
```

## CRITÉRIOS DE QUALIDADE

### A thumbnail DEVE:
- ✅ Ter o MESMO personagem das outras thumbnails
- ✅ Ser legível em tamanho pequeno (mobile)
- ✅ Gerar curiosidade imediata
- ✅ Comunicar o tema do vídeo claramente
- ✅ Ter alto contraste visual
- ✅ Ser consistente com o estilo do canal

### A thumbnail NÃO DEVE:
- ❌ Mudar o personagem
- ❌ Ter texto longo ou confuso
- ❌ Ser poluída visualmente
- ❌ Ter baixo contraste
- ❌ Mentir sobre o conteúdo (clickbait enganoso)
- ❌ Ter cores no fundo (deve ser P&B)

## PROCESSO DE CRIAÇÃO

### Passo a Passo
1. **Ler o roteiro** e identificar o conceito principal
2. **Definir o ângulo**: Revelação? Pergunta? Estatística?
3. **Escolher a expressão** do personagem apropriada
4. **Escrever o texto** de 2-3 palavras máximo
5. **Definir cores** baseadas no mapa de cor-conceito
6. **Criar o prompt** usando o template apropriado
7. **Gerar 2-3 variações** e testar
8. **Validar** em tamanho pequeno (simular mobile)

### Validação Mobile
```bash
# Reduzir para tamanho mobile e testar legibilidade
ffmpeg -i thumbnail.jpg -vf "scale=200:-1" thumbnail_mobile_test.jpg
```

## TESTING & A/B TESTE

### Variar para Testar:
1. **Expressão do personagem**: Testar 2-3 variações
2. **Texto**: Testar diferentes palavras-chave
3. **Cor do texto**: Testar branco vs vermelho vs amarelo
4. **Composição**: Testar personagem mais central vs descentrado
5. **Presença de elementos extras**: Com vs sem setas/símbolos

### Métricas para Acompanhar:
- CTR (Click-Through Rate)
- Tempo de visualização
- Engajamento (likes vs views)

## REFERÊNCIA DE ESTILO

### Manter Consistência com:
- Personagem definido no meta-prompt de imagem do personagem
- Paleta de cores do canal
- Estilo visual das cenas (P&B + elemento colorido)
- Tom do canal (analítico, profundo, não sensacionalista barato)

## 🏆 MELHOR TTS LOCAL: Kokoro-82M

Instalação:
pip install kokoro-onnx soundfile

Código Python:
from kokoro_onnx import Kokoro
import soundfile as sf

kokoro = Kokoro("kokoro-v1.0.int8.onnx", "voices-v1.0.bin")
samples, sample_rate = kokoro.create(
    "Kokoro is an open-weight TTS model with only 82 million parameters.",
    voice="af_heart",
    speed=1.0,
    lang="en-us"
)
sf.write("output.wav", samples, sample_rate)

ESPECIFICAÇÕES:
- Qualidade: ⭐⭐⭐⭐⭐ Excelente para inglês
- Hardware: CPU 4GB RAM (roda em Raspberry Pi!)
- Tamanho: 80MB (quantizado), 300MB (full)
- Velocidade: Real-time em CPU
- Licença: Apache 2.0 (uso COMERCIAL permitido)
- Vozes: af_heart, af_bella, af_nicole, af_sarah, af_sky, am_michael, am_adam

Prós:
+ Extremamente leve (80MB)
+ Rápido em qualquer hardware
+ Licença comercial livre
+ Sem custos de API
+ Roda em edge/embedded

Contras:
- Apenas inglês (não tem português)
- Menos expressivo que ElevenLabs

Links:

## 🏆 MAIS GENEROSO TTS CLOUD: Amazon Polly

Instalação:
pip install boto3

Código Python:
import boto3

polly = boto3.client('polly', region_name='us-east-1')
response = polly.synthesize_speech(
    Text="Hello! This is Amazon Polly text-to-speech.",
    OutputFormat="mp3",
    VoiceId="Joanna"
)
with open('polly.mp3', 'wb') as f:
    f.write(response['AudioStream'].read())

ESPECIFICAÇÕES:
- Limite: 5 MILHÕES caracteres/mês (~83 horas!) + 1 milhão neural

## 🏆 MAIS RÁPIDO LLM: Groq (Llama 3)

Instalação:
pip install groq

Código Python:
from groq import Groq

client = Groq(api_key="gsk_...")

chat = client.chat.completions.create(
    messages=[{
        "role": "user",
        "content": "Write a 60-second YouTube script about AI in 2026."
    }],
    model="llama-3.3-70b-versatile",
    temperature=0.7,
    max_tokens=2048
)

print(chat.choices[0].message.content)

ESPECIFICAÇÕES:
- Limite: 30 requests/minuto (free tier)
- Qualidade: ⭐⭐⭐⭐⭐ (Llama 3.3 70B)
- Velocidade: ~500 tokens/segundo! (O MAIS RÁPIDO)

- Requer CC: ❌ Não

Prós:
+ Mais rápido que qualquer outro

Contras:
